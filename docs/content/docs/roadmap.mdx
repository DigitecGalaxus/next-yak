## The Roadmap

This list is not exhaustive and not ordered by priority. It is a rough outline of what we plan to work on next.

### Completed

- Rsbuild 2 support with the next-yak/rsbuild plugin (added in v9.5.0)
- Storybook addon with Vite & Webpack support (added in v9.4.0)
- Cross-file imports working across all bundlers (v9.1.0+)
- Vite support including Vite 8 with OXC/Rolldown (added in v9.1.0)
- Turbopack support (added in v8.0.0)
- Async RSC Yak Theme Contexts (added in v8.0.0)
- Native CSS transpilation mode (added in v5.6.0)

### Planned

- Support for other runtimes (e.g. Qwik or Solid)
- More efficient runtime & code transformation
- Deduplicate styles
- Add more and better examples
